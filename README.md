# Task Tracker (Django)

## Опис
Навчальний Django-проєкт для трекінгу завдань.
Користувач може створювати задачі, переглядати список задач
та відкривати детальний перегляд кожної задачі.

Проєкт створений для вивчення:
- Django
- Class-Based Views
- Django Forms
- Роботи з моделями та шаблонами

## Технології
- Python 3
- Django
- SQLite
- HTML
